# Source Verification and Authenticity Log

## Method

Claims are labeled by source quality:

- **High confidence**: official project/vendor docs or primary repos
- **Medium confidence**: strong community sources with cross-checks
- **Low confidence**: community-only, fast-moving, or unverified in current session

## Integrity Rules

1. No claim without URL reference.
2. Any destructive operation requires two-source confirmation.
3. If source confidence is low, mark feature as research-only.
